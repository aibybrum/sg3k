Configuration Details

**Elevation will be displayed in feet
ParameterDescription
Dropzone Elevation
Type: float
Default: None		 If dropzone_elevation is set to None, the elevation will be dynamically set to the last point in the dataset.
Example: Elevation for MOORSELEEUHH 20.12
Speed Metric
Type: string
Default: km/u		 Determines the unit of speed measurement.
Distance Metric
Type: string
Default: m		 Determines the unit of distance measurement.
Toggle Search
Type: boolean
Default: True		 If True, toggle search will be displayed in the visualizations.
Front Riser
Type: boolean
Default: False		 If True, front riser will be displayed in the visualizations.
Point After Initiation
Type: float
Default: 3		 Time in seconds for a point to appear after a front riser or initiated turn.
Set to None if not wanted to display.
Pattern Elevations
Type: dict
Default: {'downwind': 1400, 'base': 1100}		 POM is automatically detected. downwind is mandatory. base can be None.
